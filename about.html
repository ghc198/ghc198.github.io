About

This is my game website, which features a LOT of games from various sources. It is based on 3kh0 lite, which comes with many of the games you see on this website. This website is not monetized, so you can enjoy these games without any ads. Usually, I'm not the one who decides to put these games on here(except for the games available on launch day). Instead, I accept requests for games because I want everyone to be able to enjoy this website. The website is open source, and you can click the link on the bottom of the page to see it.
